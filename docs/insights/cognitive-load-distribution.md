## The Problem: LLM Cognitive Overload

Using LLMs without proper architecture creates mental strain:

- Tracking multiple prompt contexts
- Context-switching between windows/sessions
- Mentally merging outputs from different prompts
- Remembering what each task was doing
- **Result:** "Headache", head heating up, exhaustion

## Why 20x Productivity

1. **Parallel execution** - Multiple lings work simultaneously
2. **Context delegation** - Coordinator stays lean
3. **Trust network** - Absorbs complexity you'd otherwise track
4. **Crystallization** - Session learnings persist (/catchup â†’ /wrap)
5. **Fire and forget** - Dispatch intent, trust shouts, collect outcomes
